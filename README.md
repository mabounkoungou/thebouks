# Picto - Personal Portfolio Free Template

#### Preview

- [Demo](https://mabounkoungou.github.io/thebouks/)



## Getting Started

1. Clone Repository

```
git clone https://github.com/mabounkoungou/thebouks.git
```

2. Install Dependencies

```
npm i
```

3. Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
